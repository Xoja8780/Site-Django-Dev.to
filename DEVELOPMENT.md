## ⚙️ Как запустить проект

```bash
# 1. Перейди в папку Django проекта
cd Dev.to-Site/Site

# 2. Активируй виртуальное окружение
venv\Scripts\activate          # Windows
source venv/bin/activate       # Mac/Linux

# 3. Установи зависимости (если первый раз)
pip install -r requirements.txt

# 4. Запусти сервер
python manage.py runserver

# 5. Открой в браузере
# http://127.0.0.1:8000/
```

## 🔑 Секреты (.env файл)

Файл `.env` лежит в папке `Site/` рядом с `manage.py`.
В гит **не попадает** — добавлен в `.gitignore`.

```
SECRET_KEY=твой_django_секретный_ключ
DEVTO_API_KEY=твой_ключ_от_dev.to
DEBUG=True
```

Где взять DEV.TO API ключ:
→ dev.to → Settings → Extensions → DEV API Keys → Generate

## 🔗 Как работает поиск

```
Пользователь вводит запрос
        ↓
forms.py (SearchForm) валидирует запрос
        ↓
views.py отправляет GET запрос к DEV.TO API
        ↓
https://dev.to/api/articles?search=запрос&per_page=30
        ↓
API возвращает JSON со списком статей
        ↓
search.html рендерит карточки статей
```

### Важные поля из API которые используем:
| Поле | Что это |
|------|---------|
| `title` | Заголовок статьи |
| `description` | Краткое описание |
| `cover_image` | Обложка (может быть null!) |
| `url` | Ссылка на статью |
| `tag_list` | Список тегов (массив) |
| `readable_publish_date` | Дата публикации |
| `reading_time_minutes` | Время чтения |
| `user.name` | Имя автора |
| `user.username` | @username автора |
| `user.profile_image_90` | Аватарка автора |

## 🐛 Известные проблемы

### 1. Поиск возвращает одинаковые результаты
**Проблема:** DEV.TO API без платного плана плохо фильтрует по `search` параметру.
Для коротких слов (`api`, `python`) возвращает популярные статьи вместо релевантных.

**Попробовать:**
- Получить платный API ключ
- Использовать параметр `tag` вместо `search` для поиска по тегам
- Попробовать сторонний поиск (Algolia, Elasticsearch)

### 2. Страница "О проекте" пустая
**Статус:** не реализована
**Что нужно:** создать `about.html` шаблон и добавить view + url
